# Customer Churn Prediction 🚀

## 📌 Overview
This project predicts whether a customer will leave a service (churn) using Machine Learning models.

## 🧠 Models Used
- Random Forest
- XGBoost

## 📊 Results
- Random Forest Accuracy: ~85%
- XGBoost Accuracy: ~88%

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

## 📈 Features
- Data preprocessing & cleaning
- Label encoding of categorical variables
- Model training & evaluation
- Feature importance visualization
- Model comparison

## 📂 Dataset
Telco Customer Churn Dataset (Kaggle)

## ▶️ How to Run
1. Open notebook in Google Colab
2. Upload dataset CSV
3. Run all cells

## 📌 Conclusion
XGBoost performed better than Random Forest, showing higher accuracy and better generalization.
